** Installation Instructions (site build configuration)

- bindir :: binary file directory; BINDIR is where the no command lands.
- texinputs :: TeX input files; TEXINPUTS is where to place the La/TeX styles/macros following the TDS.
- lib3 :: Noweb 3 library; LIB3 is where Noweb 3 scripts and configuration files go.

Noweb 3 is almost ready to come out of alpha and become beta software. The
essential things are to get /new manual pages/ and /better configuration/.

To volunteer for one of these tasks, or for something else you think needs
doing, send mail to [[mailto:nr@eecs.harvard.edu][Norman Ramsey.]]

The tasks are in rough order of priority.

| port     |        |            | Write porting instructions.                                                                                                                                                                       |
